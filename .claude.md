# BlinkTracking - Documentação do Projeto

## Visão Geral

Sistema de análise de piscadas oculares desenvolvido com Next.js 14, TypeScript e MediaPipe. O projeto permite captura, processamento e análise detalhada de movimentos das pálpebras para pesquisa oftalmológica.

## Funcionalidades Implementadas

### 1. Detecção Automática de FPS ✅
**Implementado:** 09/12/2025

Sistema detecta automaticamente o FPS do vídeo ao processar CSV:
- Suporta: 24, 30, 60, 120 FPS
- Heurística baseada em densidade de frames por minuto
- Notificação visual do FPS detectado

**Localização:** `app/analise/estatisticas/page.tsx:777-818`

```typescript
const totalFrames = parsedData[parsedData.length - 1].frame - parsedData[0].frame + 1;
const framesPerMinute = totalFrames / (estimatedDuration / 60);

if (framesPerMinute > 6000) fps = 120;
else if (framesPerMinute > 3000) fps = 60;
else if (framesPerMinute > 1500) fps = 30;
else fps = 24;
```

**Arquivos modificados:**

### 3. Novas Métricas de Piscadas ✅
**Implementado:** 09/12/2025

#### Velocidade Média de Piscadas Incompletas
Calcula separadamente a velocidade apenas das piscadas incompletas.

```typescript
if (!blink.complete) {
  totalIncompleteVelocity += velocity;
  incompleteVelocityCount++;
}
```

**Interface:** `BlinkMetrics.avgIncompleteVelocity`

#### Média das Amplitudes Máximas
Média de todas as amplitudes máximas detectadas ao longo do vídeo.

```typescript
maxAmplitudes.push(amplitude);
const avgMaxAmplitude = maxAmplitudes.reduce((a, b) => a + b, 0) / maxAmplitudes.length;
```

**Interface:** `BlinkMetrics.avgMaxAmplitude`

## Estrutura de Dados

### CSV de Pontos Faciais (MediaPipe)

**Formato:** 66 colunas
- `frame, method`
- `right_upper_1_x, right_upper_1_y, ..., right_upper_7_y` (14 cols)
- `right_lower_1_x, right_lower_1_y, ..., right_lower_9_y` (18 cols)
- `left_upper_1_x, left_upper_1_y, ..., left_upper_7_y` (14 cols)
- `left_lower_1_x, left_lower_1_y, ..., left_lower_9_y` (18 cols)

**Documentação:** `public/docs/MEDIAPIPE_POINTS_MAPPING.md`

### Interface BlinkMetrics

```typescript
interface BlinkMetrics {
  // Contagem
  totalBlinks: number
  completeBlinks: number
  incompleteBlinks: number

  // Velocidade
  avgVelocity: number
  avgIncompleteVelocity: number  // ✨ Novo
  velocityBasedBlinks: number

  // Amplitude
  rba: number
  avgMaxAmplitude: number  // ✨ Novo

  // Timing
  avgDuration: number
  avgBlinkInterval: number
  blinkRate: number
  blinks60to120: number
  blinks120to180: number

  // Fenda (antes Fissura)
  verticalFissure: number
  horizontalFissure: number

  // Primeiras piscadas
  firstCompleteBlink: {
    ecp: number  // Eye Closing Phase
    cdp: number  // Complete Closure Duration Phase
    eop: number  // Eye Opening Phase
    ibl: number  // Inter Blink Latency
  }
  firstIncompleteBlink: { ... }

  // Distribuição
  distribution: {
    complete: number
    incomplete: number
    other: number
  }
}
```

## Padrões de Código

### Detecção de Piscadas

**Algoritmo baseado em threshold adaptativo:**

```typescript
const maxDistance = Math.max(...distances);
const minDistance = Math.min(...distances);

// 60% do range para detectar piscada
const blinkThreshold = minDistance + (maxDistance - minDistance) * 0.6;

// 30% do range para piscada completa
const completeBlinkThreshold = minDistance + (maxDistance - minDistance) * 0.3;
```

### Detecção por Velocidade (MAD)

```typescript
const mad = calculateMAD(velocities);
const velocityThreshold = 3 * mad;

if (velocity > velocityThreshold) {
  // Piscada detectada
}
```
